# HTML-CSS

## Featured Projects

### 🌿 The Digital Garden Portfolio
A custom-built personal website for **Sitnoor** that utilizes advanced CSS techniques:
* **Glassmorphism:** Uses `backdrop-filter` for a frosted glass effect on the hero text and navbar.
* **Dynamic Media:** Features a high-definition forest background video that autoplays silently.
* **Interactivity:** Includes a grayscale-to-color image filter and a scaling hover effect on typography.
* **Layouts:** Implements a fixed navigation menu and a color-coded two-column grid.

### 🎬 Bootstrap Movie Theater
A project focused on rapid prototyping using the **Bootstrap 5** framework.
* **Dynamic Components:** Uses Card Decks to organize movie content (Kong, Ghost in the Shell, etc.).
* **Navigation:** Built-in anchor points for seamless single-page navigation.
* **Badges:** Utilizes Pill Badges to show varied movie showtimes.

## Technical Skills Applied
- **HTML5:** Semantic structures, Video embedding, Tables.
- **CSS3:** Flexbox layouts, Image Overlays, Transitions, Filter effects.
- **Frameworks:** Bootstrap 5 (Responsive Grids, Components).
- **Git:** Version control and repository management.
